# Inverse transformation (quantile) for EGPD

Inverse transformation (quantile) for EGPD

## Usage

``` r
q.G(u, type = 1, prob = NA, kappa = NA, delta = NA)
```

## Arguments

- u:

  values in (0,1)

- type:

  integer 1-6

- prob:

  mixing probability (type 6)

- kappa:

  shape parameter

- delta:

  shape parameter (types 4-6)

## Value

Inverse-transformed values
