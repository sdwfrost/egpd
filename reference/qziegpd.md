# Quantile function of the Zero-Inflated Extended GPD

Quantile function of the Zero-Inflated Extended GPD

## Usage

``` r
qziegpd(
  p,
  pi = NA,
  prob = NA,
  kappa = NA,
  delta = NA,
  sigma = NA,
  xi = NA,
  type = 1
)
```

## Arguments

- p:

  probabilities

- pi:

  zero-inflation probability

- prob:

  mixing probability (type 6)

- kappa:

  shape parameter for G transformation

- delta:

  shape parameter for G transformation (types 4-6)

- sigma:

  GPD scale parameter

- xi:

  GPD shape parameter

- type:

  integer 1-6 specifying G type

## Value

Quantile values
