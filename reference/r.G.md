# Random generation from transformation G

Random generation from transformation G

## Usage

``` r
r.G(
  n,
  prob = NA,
  kappa = NA,
  delta = NA,
  type = 1,
  unifsamp = NULL,
  direct = FALSE
)
```

## Arguments

- n:

  number of samples

- prob:

  mixing probability (type 6)

- kappa:

  shape parameter

- delta:

  shape parameter (types 4-6)

- type:

  integer 1-6

- unifsamp:

  optional uniform samples

- direct:

  logical for type 6

## Value

Random samples
