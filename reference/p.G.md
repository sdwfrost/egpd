# Transformation CDF for EGPD

Transformation CDF for EGPD

## Usage

``` r
p.G(u, type = 1, prob, kappa, delta)
```

## Arguments

- u:

  values in (0,1)

- type:

  integer 1-6 specifying the transformation type

- prob:

  mixing probability (type 6)

- kappa:

  shape parameter

- delta:

  shape parameter (types 4-6)

## Value

Transformed probabilities
